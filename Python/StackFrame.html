StackFrame class¶

Represents a stack frame in the target process.

class dbgscript.StackFrame¶
StackFrame.frame_number → int¶

Get the frame number of this stack frame.

StackFrame.instruction_offset → int¶

Get the virtual address of the frame’s current instruction.

StackFrame.get_locals() → tuple of TypedObject¶

Get the local variables in the stack frame.

StackFrame.get_args() → tuple of TypedObject¶

Similar to get_locals() except return the arguments only.
